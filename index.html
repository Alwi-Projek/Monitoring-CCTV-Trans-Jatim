Input / Update Monitoring

Data dengan kombinasi tanggal + bus yang sama akan diperbarui. Pilih koridor dulu supaya daftar bus lebih pendek dan lebih cepat dicari.
Ide percepatan monitoring harian:
Saat belum ada perubahan dari vendor, gunakan fitur Copy Filter ke Tanggal Baru atau Copy Semua Tanggal. Sistem akan menyalin data tanggal sebelumnya ke tanggal berikutnya tanpa input ulang satu per satu. Jika ada bus yang sudah diperbaiki, Anda tinggal edit unit tersebut saja.

Copy Monitoring Cepat

Fitur ini untuk kondisi data harian masih sama. Salin satu tanggal ke tanggal lain, dengan opsi hanya koridor tertentu atau seluruh data.

Filter & Rekap

Sekarang ada filter tindakan Butuh Perbaikan dan Berfungsi Normal. Tabel tetap dikelompokkan per koridor.
