Input / Update Fasilitas

Pilih koridor dulu agar pemilihan bus lebih cepat. Tanggal disimpan otomatis sebagai update terakhir.

Logika tanggal otomatis:
- Saat status menjadi Butuh Perbaikan, sistem mengisi Tanggal Mulai Gangguan jika sebelumnya kosong.
- Saat status menjadi Berfungsi Normal, sistem mengisi Tanggal Selesai.
- Tabel utama menampilkan Last Update, bukan monitoring harian.
